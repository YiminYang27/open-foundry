## Speaker Selection

Rules:
- Pick the agent whose perspective is MOST needed right now
- If an agent just made a claim, prefer a contrasting perspective
- Going in circles (same points 3+ times) -> break impasse or declare CONSENSUS
- CONSENSUS only when all agents have spoken AND recent turns show convergence

Evidence quality gating:
- If an agent states a number (price, rate, percentage, date) WITHOUT citing a
  source or search result, immediately pick critical_analyst or quant_engineer
  to challenge it. Unsourced data must not go unchallenged.
- If multiple agents are building on an unverified data point, break the chain
  and pick an agent to verify the foundational claim first.

Financial discussion priorities:
- In the first third of the discussion, ensure EVERY domain gets at least one turn:
  macro environment, geopolitical risk, asset-specific fundamentals, institutional views
- When an agent makes a quantitative claim (price target, probability, ratio),
  pick an agent who can validate or challenge the underlying assumptions
- Before declaring CONSENSUS, verify that the discussion has produced:
  (a) a base case with specific numbers backed by cited sources,
  (b) at least one bull/bear scenario with trigger conditions,
  (c) key risk events identified with dates
- If agents are agreeing without quantitative grounding, pick the agent most
  likely to demand evidence (critical_analyst or quant_engineer)
- Prefer alternating between "thesis builders" (macro_economist, gold_analyst,
  geopolitical_analyst) and "thesis testers" (critical_analyst, quant_engineer,
  sellside_strategist) to maintain analytical rigor

## Closing Summary

Write a closing summary (300-600 words) structured as:

1. **Consensus view**: the base case price outlook with specific numbers,
   timeframe, and the key assumptions behind it
2. **Bull case**: trigger conditions, probability assessment if discussed,
   and which agents supported it
3. **Bear case**: trigger conditions, probability assessment if discussed,
   and which agents supported it
4. **Unresolved disagreements**: where agents diverged and why -- name the
   specific agents and their positions
5. **Key risk events**: calendar of upcoming events that could shift the outlook
   (Fed meetings, data releases, geopolitical flashpoints)
6. **Monitoring indicators**: what data points to watch for scenario shifts

Reference specific agents' contributions and quantitative claims.
Flag any claims that were asserted but not challenged.
ASCII only.
